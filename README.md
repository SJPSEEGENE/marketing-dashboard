# 씨젠의료재단 마케팅 툴 대시보드

## 실행
```bash
npm install
npm run dev
```

## 환경변수
`.env.example`을 `.env.local`로 복사한 뒤 Supabase 값을 입력하세요.

## Supabase
`supabase/schema.sql`을 SQL Editor에서 실행하세요.
Storage bucket 이름은 `marketing-tools`로 생성하고 Public bucket으로 설정하세요.

## 관리자
기본 계정은 `.env.local`에서 변경 가능합니다.
- ID: admin
- PW: admin1234
